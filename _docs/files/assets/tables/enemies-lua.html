Enemies

Enemies are referenced in tiled by there numerical ID.  Remember to update the ID number or the Enemy will be overwritten with the new value.

Properties

nameName of enemy
imageLocation of image
widthWidth of image
heightHeight of image
statsEnemy stats like hp & mp Not currently implemented.
expAmount of exp given Not currently implemented.
goldAmount of gold given Not currently implemented.
dropsItems/Equipment given Not currently implemented.

Example code

[1] = {
    name = 'Water Dragon',
    image = 'assets/img/water_dragon.gif',
    width = 128,
    height = 128,
    stats = {
        level = 1,
        hp = 275,
        hpmax = 280,
        mp = 60,
        mpmax = 70,
        exp = 50,
        expmax = 100,
        def = 20,
        mdef = 10,
        spd = 10,
    },
    exp = 100,
    gold = 99,
    drops = {
        equipment = {1,2,3,4},
        items = {1,2}
    }
}
