Heroes

Base stats for when the hero joins your party.  This table is copied directly to the STATE variable Stats will me modified automatically by the game.

Properties

nameName of hero
imageLocation of image
widthWidth of image
heightHeight of image
statsHero stats like hp & mp
activeDetermines if a character is in your party.

Example code

[1] = {
    name = 'Hero 1',
    image = 'assets/img/water_dragon.gif',
    width = 128,
    height = 128,
    stats = {
        level = 1,
        hp = 275,
        hpmax = 280,
        mp = 60,
        mpmax = 70,
        exp = 50,
        expmax = 100,
        def = 20,
        mdef = 10,
        spd = 10,
    },
    active = true
},
